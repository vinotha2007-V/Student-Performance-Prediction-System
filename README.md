# 🎓 Student Performance Prediction System

## 📌 Overview
This project predicts student academic performance using Machine Learning.

## 🚀 Features
- Predict student scores
- Identify low-performing students
- Flask web interface

## 🧠 ML Model
- Linear Regression

## 📊 Input Features
- Study Hours
- Attendance
- Previous Score
- Assignments Completed
